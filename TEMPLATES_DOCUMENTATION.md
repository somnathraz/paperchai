## ğŸ”§ Technical Implementation

All templates use the `TemplateProps` interface:
```typescript
interface TemplateProps {
  preview?: boolean;
  modalPreview?: boolean;
  mockData?: {
    businessName?: string;
    invoiceNumber?: string;
    clientName?: string;
    total?: string;
    items?: Array<{ name: string; amount: string }>;
    subtotal?: string;
    tax?: string;
    notes?: string;
    paymentTerms?: string;
  };
  previewDark?: boolean;
}
```

Templates are registered in `components/invoices/templates/registry.tsx` and can be rendered using:
```typescript
import { renderTemplate } from "./templates/registry";
renderTemplate(templateSlug, props);
```
